# 3D 孔标注工具 (Hole Annotator Tool)

自动识别 STEP 模型中的螺纹孔、销孔、沉孔，生成可旋转的 3D 标注查看器。

## 功能

- 上传 STEP 文件自动分析（支持装配体、焊件、单零件）
- 装配结构树展示，可勾选要标注的零件
- 焊件模式：将整个焊接组件作为一个整体标注
- 识别螺纹孔（M3~M20）、销孔（H7）、通孔、沉孔
- 自动区分内孔与外圆柱面（凸台/销钉不会被误标）
- 标注点自动定位到零件外表面
- 生成交互式 3D HTML 查看器（可旋转/缩放，完全离线）
- 标注可拖动调整位置，双击可修改内容
- 布局自动保存

## 给同事分发

### 准备分发包（你只需做一次）：

1. 在此目录右键 → "在终端中打开"
2. 运行：`powershell -ExecutionPolicy Bypass -File setup_embedded_python.ps1`
3. 等待完成（下载 Python + 安装依赖，约 5 分钟）
4. 完成后把整个 `hole_annotator_tool` 文件夹压缩发给同事

### 同事使用：

1. 解压文件夹到任意位置
2. 双击 `start.cmd`
3. 浏览器自动打开
4. 上传 STEP 文件 → 自动生成 3D 标注

**无需安装任何软件，解压即用。**

## 操作说明

- 鼠标左键：旋转模型
- 鼠标右键：平移
- 滚轮：缩放
- 拖动标注：调整位置
- 双击标注：修改内容（如改为 M8-6H）
- "保存标注布局"按钮：保存调整

## 技术栈

- CadQuery + OpenCascade：STEP 解析和孔识别
- three.js：3D 渲染
- FastAPI：Web 服务
- 内置 Python 3.12（无需系统安装）

无需 FreeCAD，无需 LLM，纯几何分析。
